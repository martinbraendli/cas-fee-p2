# Rezept-Single-Page-App

## Anforderungen

Produkt Modul


Editor Modul (CUG)

- Rezepte Erfassen / Löschen / ändern


Rating Modul 
- Rezepte bewerten / Eigene Ratings wieder rückgängig machen.

User Modul
- User erfassen und verwalten
- Bestellungs-Verlauf

Security Modul
- Login / Logout
- Optional: Auch anonymer Check Out möglich.
